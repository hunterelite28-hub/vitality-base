FACETED GEMS (v2 of Direction A) Each corner gem is a flat-SVG faceted polyhedron with the canonical library shape and glyph: Train uses icosahedron + BAR, Fuel uses sphere + DROP, Peak uses octahedron + BOLT, Mind uses dodecahedron + HEX, Brand uses icosahedron + V, Finance uses dodecahedron + ASCEND in amber to match its warm palette.

ANIMATIONS Gem body breathes (scale 1 to 1.04, 3.6s loop). Halo glow pulses behind it (4.4s). Whole slot floats up and back (9s slow drift). Inner glyph cross-fades between the module glyph and the brand V every 6s (idle morph pattern from the real SectionGem). Glyph adds a subtle flicker on top (2.8s).

WHY FLAT SVG INSTEAD OF THE REAL 3D CRYSTAL The library renders the real Three.js crystal at 280px default. Below 240px the glass goes muddy. Corner gems at 56px would not read in 3D, so this is a flat simulation tuned to look like the same crystal language. If we want a real 3D gem somewhere on the dashboard later, the only spot it would fit cleanly is centered on the Train hero tile at ~240px.
